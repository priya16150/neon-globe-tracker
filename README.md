# ✨ Hand Canvas – Interactive Hand Tracking Experience

An advanced interactive hand-tracking playground built with **MediaPipe Hands** and **HTML5 Canvas**. Control multiple creative simulations using real-time hand gestures including elastic strings, stretchable bands, a rotating wireframe globe, and gesture-based digital painting.

## 🚀 Features

* 🖐️ Real-time hand tracking with MediaPipe
* 👋 Supports up to 2 hands simultaneously
* 🎨 Multiple interactive modes
* 📷 Live camera feed with hand landmark visualization
* ✨ Neon-inspired futuristic interface
* ⚡ Smooth 60 FPS canvas rendering
* 📱 Responsive fullscreen experience
* 🎭 Real-time gesture detection

## 🎮 Available Modes

### 〰 Strings Mode

* Interactive elastic string simulation
* Finger movements push and pull virtual strings
* Physics-based rope dynamics
* Colorful glowing strands

### ⟳ Elastic Mode

* Stretchable elastic bands
* Pinch gesture to grab and pull bands
* Dynamic tension visualization
* Realistic elastic behavior

### ⬡ Globe Mode

* Interactive rotating wireframe globe
* Hand movement controls globe rotation
* Futuristic holographic appearance
* Dynamic lighting effects

### ✦ Paint Mode

* Draw in mid-air using hand gestures
* Multi-color fingertip trails
* Neon glow brush effects
* Gesture-controlled drawing system

## 🚀 Installation

### Clone Repository

```bash
https://github.com/priya16150/neon-globe-tracker
```

### Open Project

```bash
cd hand-canvas
```

### Run Locally

Using VS Code Live Server:

```bash
Right Click → Open with Live Server
```

Or using Python:

```bash
python -m http.server 8080
```

Open:

```text
http://localhost:8080
```
